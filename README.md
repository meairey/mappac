# mappac
### The `mappac` library is a tool which creates current maps for global surface currents. It uses gridded data from the HYCOM database and returns both data cubes as well as heatmaps. 

## Installation 
    git clone https://github.com/meairey/mappac.git
    cd mappac/
    pip install .

## Dependencies
    conda install matplotlib numpy iris cartopy
